Вопросы

  1. В каком регистре лучше писать HTML-код?
  2. Как убрать синюю рамку вокруг картинки-ссылки?
  3. Обязательно ли писать alt в <img>?
  4. Что такое html-entities?
  5. Как убрать подчеркивание для всех ссылок на странице?
  6. Что делает свойство clear?
  7. Есть ячейка таблицы, в ней 3 дива, в каждом диве – по слову. На первый див применен стиль float: left; на третий – float: right. Что нужно применить на второй див, чтобы все три надписи/дивы были в одной горизонтальной строке, то есть первый имел бы выравнивание по левому краю, третий – по правому, а второй – по центру?
  8. Перечислить по пять знакомых вам псевдоклассов и псевдостилей с указанием их функционала.

Ответы

  1. В любом регистре, т.к. html - нерегистрозавсимый, но принято писать с маленькой буквы (html)
  2. Убрать рамку можно с помощью правила css - border: none;
  3. Атрибут alt у тега <img> предназначен для отображения альтернативного текста в случае, когда картинка не загружается. Его использование основнно на здравом смысле + его использование обязательно по стандарту.
  4. html-entities - спецсимволы (& lt ; - <) (& # 8209; - ‑) (O& # 770; - Ô)
  5. Задать css-правило для тега <a> - text-decoration:none;
  6. Свойство clear CSS указывает, может ли элемент быть рядом с плавающими floating элементами, которые предшествуют ему или должны быть перемещены вниз (очищены) под ними.

  7. Ко второму диву надо применить правило такое же, как и к первому (float:left;)

  8. Псевдоклассы css

    • :link - правила для непосещенных ссылок
    • :visited - правила для посещенных ссылок
    • :hover - правила для наведение на элементы
    • :active - правила для нажатия на ссылки
    • :first-child - правила для первого элемента из группы таких же элементов
